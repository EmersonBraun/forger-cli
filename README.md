# Forger CLI
A CLI to generate files

### Prerequisites

* [Node.js &gt; 10](https://nodejs.org/en/)
* [NPM](https://www.npmjs.com/)

### Installing
Install globaly using NPM
```
npm i -g @emersonbraun/forge
```

### Usage

Call the CLI
```
forger
```
